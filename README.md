# .github

This is the [BLIT386](https://github.com/blit386) GitHub organization's special `.github` repository. It holds the
org-wide profile and the default community-health files (code of conduct, and similar) that other repos in the
organization fall back to when they do not define their own.

- Org profile: [`profile/README.md`](profile/README.md) - renders on [github.com/blit386](https://github.com/blit386)
- Default code of conduct: [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md)

Looking for the engine itself? Start at [blit386/blit386](https://github.com/blit386/blit386) or the docs at
[blit386.dev](https://blit386.dev).
